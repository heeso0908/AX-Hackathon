# CSRD·CBAM 맥락 VoC 인사이트

## Insight 1. CBAM 관련 문의·불만 집중, 확정 시행 리스크 구간 진입

- 관찰: CBAM/CSRD/Scope3 규제 관련 VoC가 전체 44건 중 19건(43.2%)이며, 그중 'CBAM' 맥락이 10건으로 가장 많다.
- 근거 데이터: 규제 관련 VoC 19건 중 urgency Medium 이상이 9건(47.4%). 대표 사례: id010(EU 세관 반려), id045(신고 마감 초과).
- 비즈니스 의미: (산업 동향: 2026년 1월 ~ (확정 기간) CBAM 인증서 구매 의무화, 본격 과금 시작) 규제 확정 시행 시점과 맞물려 신고 절차·인증서·제출 양식 관련 혼란이 CS 부담과 계약 리스크(반려, 마감 지연)로 이어질 수 있다.
- 권장 액션: 'CBAM' 신고 절차/제출 양식에 대한 표준 가이드를 우선 제작하고, EU 세관 반려·마감 초과처럼 계약 리스크로 직결되는 건은 컨설팅팀이 개별 대응 SLA를 적용한다.
- 관련 VoC 예시: id 006("CBAM 인증서 구매 절차를 단계별로 설명하는 가이드 문서를 만들어 주시면 좋겠습니다."), id 010("카본링크에서 출력한 CBAM 보고서가 EU 세관에서 반려됐습니다."), id 013("CSRD 이중 중요성(double materiality) 평가 방법론을 플랫폼 내에서 지원...")

## Insight 2. 제품 개선 신호 집중: 'Scope 3/공급망 데이터'

- 관찰: 불만·기능 요청 중 제품/기능/데이터 신뢰성과 직결되는 VoC가 15건(34.1%)이며, 'Scope 3/공급망 데이터' 주제가 5건으로 가장 많다.
- 근거 데이터: 'Scope 3/공급망 데이터' 관련 5건이 불만·기능 요청 후보 15건 중 33.3%를 차지한다.
- 비즈니스 의미: 협력사 수백 개의 Scope 3 데이터를 수동으로 입력해야 하는 구조는 데이터 신뢰성 저하와 CS 반복 문의의 근본 원인이 될 수 있다.
- 권장 액션: 협력사 데이터 일괄 요청·자동 수집 기능(대량 발송, 자동 리마인드)을 우선 검토하고, 200개 협력사 규모의 고객(id008)을 베타 대상으로 삼는다.
- 관련 VoC 예시: id 005("Scope 3 카테고리 15번(투자) 계산이 너무 복잡합니다."), id 008("공급망 2차 협력사 데이터 입력 기능이 너무 불편합니다."), id 018("공급업체 탄소 데이터 공유 요청을 일괄 발송하는 기능이 필요합니다.")

## Insight 3. 단순 정보 확인형 문의의 FAQ/가이드 전환 여지

- 관찰: 전체 VoC 44건 중 15건(34.1%)이 단순 정보 확인형 문의(일반 문의 또는 가이드 요청)이며, 'CBAM 신고/인증서' 주제가 7건으로 가장 많다.
- 근거 데이터: FAQ/가이드화 후보 15건 중 'CBAM 신고/인증서' 관련 7건(46.7%)이 반복되는 절차성 질문이다.
- 비즈니스 의미: (회사 컨텍스트: 리뉴어스랩의 고객은 대부분 탄소규제 대응을 처음 경험하는 실무자들입니다. 이들은 규제 해석, 데이터 입력 방식, 신고 기한, 플랫폼 사용법 등 다양한 채널로 질문과 불만을 제기합니다.) 반복되는 절차 문의를 지식베이스로 전환하면 신규 CS 담당자의 도메인 학습 부담과 1건당 응대 시간을 함께 줄일 수 있다.
- 권장 액션: 'CBAM 신고/인증서' 주제를 포함한 상위 문의 패턴을 카본링크 헬프센터 FAQ로 우선 전환하고, 반복 빈도가 높은 항목은 챗봇/자동 응답 초안을 마련한다.
- 관련 VoC 예시: id 001("CBAM 신고 양식 제출 기한이 정확히 언제인가요?"), id 006("CBAM 인증서 구매 절차를 단계별로 설명하는 가이드 문서를 만들어 주시면 좋겠습니다."), id 016("CBAM 전환 기간이 끝나고 2026년 1월부터 인증서를 실제로 구매해야 한다는 게 맞나요...")

## Insight 4. OEM/1차 협력사의 고긴급도 불만 - 이탈 위험 신호

- 관찰: 불만 12건 중 OEM/1차 협력사에서 발생한 urgency High 건이 3건 확인된다.
- 근거 데이터: id 010, 027, 045 — EU 세관 반려·인증 데이터 내보내기 실패·신고 마감 초과 등 계약·업무 리스크로 직결되는 사례.
- 비즈니스 의미: (회사 컨텍스트: 이탈 위험 감지: 처리 지연된 불만 VoC는 계약 해지 예측 변수) 고가치 협력사의 고긴급도 불만은 처리 지연 시 계약 갱신율(NRR) 저하나 이탈로 이어질 수 있는 선행 신호다.
- 권장 액션: 해당 건은 SLA 24시간 이내 전담 대응을 적용하고, 처리 완료 후 CSM이 후속 만족도 체크를 진행해 이탈 신호를 조기에 해소한다.
- 관련 VoC 예시: id 010("카본링크에서 출력한 CBAM 보고서가 EU 세관에서 반려됐습니다."), id 027("ISO 14064 인증 제출용 데이터 내보내기가 안 됩니다."), id 045("CBAM 신고 마감일이 지났는데 연장 신청이 가능한지 EU 세관에 문의하는 방법을 모르겠습...")

## Insight 5. 칭찬 VoC 중 세일즈 레퍼런스 활용 가능 소재

- 관찰: 칭찬 VoC 8건(18.2%) 중 온보딩 응대, 업무 자동화, 보고서 품질 개선 등 정량적 성과가 언급된 사례가 3건 확인된다.
- 근거 데이터: id 024, 034, 039 — 업무 시간 절감·오류 감소·배출량 감소 파악 등 구체적 수치가 포함된 후기.
- 비즈니스 의미: (회사 컨텍스트: 영업 레퍼런스 발굴: 칭찬 VoC는 세일즈 케이스 스터디와 마케팅 콘텐츠 소스) 정량적 성과가 포함된 칭찬 후기는 별도 인터뷰 없이도 세일즈 케이스 스터디·마케팅 콘텐츠의 1차 소재로 바로 활용할 수 있다.
- 권장 액션: 위 사례를 고객 동의 하에 케이스 스터디로 재구성해 세일즈/마케팅팀에 전달하고, 레퍼런스 콜이 가능한 고객인지 CSM이 확인한다.
- 관련 VoC 예시: id 024("작년에 비해 탄소 배출 집계 오류가 많이 줄었습니다."), id 034("공급망 탄소 데이터 수집 자동화가 도입된 이후로 업무 시간이 절반 이상 줄었습니다."), id 039("지난 분기 대비 탄소배출량이 15% 감소했습니다.")
